# Image Binary Matrix

A small application to fetch an image mask based on a specified color range and generate a binary matrix corresponding to the mask.

```shell
make build
```

P.S. The first time launch may last long time

```shell
make run
```

## Technologies

- GLEW
- GLFW
- ImGui
- OpenCV
